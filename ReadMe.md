**Description:**  
Penguin Hub is a full-stack web application that creates a fun and interactive world where penguins can build profiles, apply for jobs, follow fitness plans, and even have criminal records.

The project demonstrates full CRUD functionality, authentication, authorization, and relational data handling using the MVC pattern.

## Getting Started

**Run Locally:**  
1. Clone the repo: `git clone [AUTH]`  
2. Install dependencies: `npm install`  
3. Run the server: `npm start`  
4. Open in browser at `http://localhost:3000`


## Technologies Used
- **Frontend:** HTML, CSS, JavaScript, EJS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB with Mongoose  
- **Other Libraries:** Method-override, bcrypt (if using authentication), dotenv


## Features
- User authentication (Sign up / Sign in)
- Penguin profiles (CRUD)
- Job listings with applications system
- Search functionality for jobs
- Gym feature:
BMI calculation
Workout plan based on weight & height

- Crime Records system:
Add, edit, delete crimes
Crime severity & punishment

- Public & protected routes
- Responsive and modern UI design
